Explicación

En este ejemplo implementamos ya la funcionalidad de saltar al contenido principal. El enlace tiene las siguientes propiedades:

  • position: absolute // para sacarlo del flujo principal.
  • clip-path: inset(50%) // recortamos el elemento para que no sea visible.
  • clip: rect(1px, 1px, 1px, 1px) // similar al anterior. Se mantiene por compatibilidad con navegadore antiguos. Requiere que el elemento tenga position absolute o fixed.
  • height: 1px
  • width: 1px
  • overflow: hidden // aquel contenido que excede el área del elemento no es mostrado.
  • white-space: nowrap // mostraría todo el contenido en una línea.

Cuando el enlace recibe el foco (al navegar por teclado), las propiedades del elemento son:

  • clip-path: inset(0%); // no recortamos el elemento
  • clip: auto
  • height: auto
  • width: auto
  • top: 0 // lo podemos colocar en la parte superior.
  • otras propiedades que nos permitan dar el estilo deseado.
