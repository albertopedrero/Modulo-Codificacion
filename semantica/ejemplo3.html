Landmarks

Esta página contiene algunos landmarks básicos

  • Main o principal
  • Footer o pie
  • Aside

Esto es el main

Aquí incluiremos el contenido principal. Puede estar organizado en secciones o artículos.

Esto es el aside

Contiene contenido tangencial, es decir, contenido relacionado con el contenido del main, pero que no puede considerarse como clave.

Footer

Aquí colocaremos, entre otros, el enlace de accesibilidad.
