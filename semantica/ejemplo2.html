Comillas diferentes según idioma

Los siguientes ejemplos muestran citas en diferentes idiomas. También se aplica un color diferente para el contenido en alemán.

El carácter utilizado para las comillas de las citas puede variar según el idioma.
The character used for quotation marks may vary depending on the language.
Das für Anführungszeichen verwendete Zeichen kann je nach Sprache variieren.

Hasta ahora hemos probado la combinación de etiquetas <blockquote> y <q>. La primera hace que el contenido aparezca con cierto espacio a izquierda y derecha. Ahora veremos la etiqueta <q> dentro de un párrafo.

Esto es un párrafo en castellano. El carácter utilizado para las comillas de las citas puede variar según el idioma. Lo mismo para valores decimales, como 7.200.

This is a paragraph in Englis. The character used for quotation marks may vary depending on the language. The same applies to decimal values, such as 7.200.

Dies ist ein Absatz auf Deutsch. Das für Anführungszeichen verwendete Zeichen kann je nach Sprache variieren. Das Gleiche gilt für Dezimalwerte wie 7.200.
